HYDRA is a YouTube comment marketing automation system managing 100-200 accounts to appear as real users. Core flow: account CSV import -> AdsPower profile -> persona -> warmup -> active -> campaign execution (10 scenarios A-J) -> like boosting -> ghost detection.

**Why:** Commercial YouTube marketing service requiring stealth, scale, and operational dashboard.

**How to apply:** All recommendations should prioritize detection evasion and operational stability. The user (seominjae) is the sole developer and operator -- code should be maintainable by one person. SQLite is the intentional DB choice for single-machine deployment.
