# HYDRA v2 Phase 3: Worker 앱 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Worker PC에서 실행되는 독립 앱을 만든다. 서버에 연결하여 태스크를 받아 실행하고, 결과를 보고하며, 자동 업데이트를 지원한다.

**Architecture:** worker/ 패키지로 분리. httpx로 서버 API 호출, AdsPower 로컬 API 연동, Playwright로 브라우저 자동화. 메인 루프: heartbeat → 태스크 fetch → 실행 → 보고.

**Tech Stack:** Python, httpx, AdsPower Local API, Playwright
